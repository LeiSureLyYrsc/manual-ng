# 新手指引

作为一名 LittleSkin 的用户，你使用 LittleSkin 的目的无非就是：

- 在 LittleSkin 上分享你制作的皮肤或披风材质
- 借助 LittleSkin 在离线登录的游戏中显示自己的皮肤或披风
- 使用 LittleSkin 作为外置登录服务并加入特定的服务器

这份手册将帮助你了解如何使用 LittleSkin，并解决在此过程中你遇到的各种问题。
